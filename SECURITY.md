# Security Policy

## Reporting a vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, email **security@bird.com** with a description of the issue, the steps to reproduce it, and the affected version. We will acknowledge your report and keep you informed of the progress toward a fix.

## Scope

This repository is a generated release mirror of the Bird Python SDK. Reports about the SDK are in scope; reports about the Bird API or platform are also welcome at the same address.
